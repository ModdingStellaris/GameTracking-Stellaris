- Fixed "Colonial Failure" anomaly event handing out a Colonization tech you already had.
- Fixed "Master's Teachings: Philosophical Mindset" edict not having a monthly influence cost.
- Fixed an issue with "The Tree" anomaly event spawning amoeba creatures that weren't considered to belong to the amoeba container country.
- Fixed an issue with the "Colonial Assassin" event not applying an opinion modifier correctly.
- Space Monster contact projects should now always be valid and available once generated
- Extradimensional Crisis should no longer spawn inside Fallen Empire borders
- Fixed tooltip for Grand Mausoleum when invalid to build
- Changed the way native reservations are placed for a colony event related to colonizing planets that have primitive natives.
- Fixed several small issues with the sublight space probe questline. Making sure that there are indicators to show what you need to do in each system and that every step is shut down properly.
- Fix for missing text when a leader gains the Engineer trait.
- Fixed an issue where the Hyacinthe questline couldn't finish.
- Potential fix for Alien Specimen Procurement chain spawning Special Projects in Fallen Empire space.
- Egalitarian empires are now unable to get leaders with slaver mandates, and egalitarian pops are less likely to vote for candidates with slaver mandate.
